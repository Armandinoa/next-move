# Commercial Support

Next Move is open source, but commercial support helps fund development and
gives users a faster path from idea to working workflow.

## Services

### Setup Sprint

For solo builders who want Next Move connected to a real project.

Includes:

- Install and run Next Move on one repository.
- Generate a roadmap and issue drafts.
- Prioritize the first implementation cycle.
- Explain the free/paid boundary for the project.

Suggested starting price: 49-99 EUR.

### Automation Sprint

For builders who want recurring project reviews.

Includes:

- Scheduled scans on a VPS, GitHub Actions, or another runner.
- Report delivery to GitHub Issues, Telegram, Discord, Slack, or email.
- Basic project-specific playbook.
- One improvement cycle using `--self-improve`.

Suggested starting price: 149-299 EUR.

### Team Pilot

For small teams with multiple repositories.

Includes:

- Multi-repo review.
- Shared roadmap format.
- Risk and monetization review.
- Suggested process for weekly project momentum.

Suggested starting price: 499+ EUR.

## What Commercial Support Does Not Include

- Guaranteed revenue.
- Legal, tax, financial, or investment advice.
- Ownership of user repositories.
- Closed-source access to the free core CLI.

## Contact

Open a GitHub issue with the `commercial-support` label, or sponsor the project
and include a short note about what you want help with.
